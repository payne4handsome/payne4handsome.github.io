1. Flow-OPD: On-Policy Distillation for Flow Matching Models#

Zhen Fang, Wenxuan Huang, Yu Zeng

现有流匹配（Flow Matching, FM）文本到图像模型在多任务对齐中面临奖励稀疏性与异构目标联合优化引发的梯度干扰两大瓶颈，导致指标“跷跷板效应”与普遍的奖励作弊现象。本文提出Flow-OPD——首个将在线策略蒸馏（On-Policy Distillation, OPD）引入FM模型的统一后训练框架。其采用两阶段对齐策略：先通过单奖励GRPO微调构建领域专用教师模型；再基于流匹配的冷启动机制初始化策略，并通过在线采样、任务路由标注与稠密轨迹级监督完成知识融合。进一步提出流形锚点正则化（MAR），利用无任务偏置的教师提供全数据监督，将生成锚定至高质量流形，显著缓解纯强化学习对齐导致的美学退化。在Stable Diffusion 3.5 Medium上，GenEval得分由63提升至92，OCR准确率由59提升至94。

2. LLMs Improving LLMs: Agentic Discovery for Test-Time Scaling#

Tong Zheng, Haolin Liu, Chengsong Huang

本文针对测试时扩展（TTS）策略依赖人工设计、搜索空间受限的问题，提出环境驱动的自动发现框架AutoTTS。其核心在于构建可解耦、反馈密集的发现环境，将TTS策略发现建模为基于预采集推理轨迹与探针信号的控制器综合问题，支持分支、继续、探针、剪枝与终止等细粒度决策，并通过β参数化与执行迹反馈提升搜索效率与可诊断性。实验表明，在数学推理基准上，自动发现的策略在准确率—计算成本权衡上显著优于强手工基线，且具备跨任务与模型规模的泛化能力；整个发现过程仅耗时160分钟、成本39.9美元。

3. Normalizing Trajectory Models#

Jiatao Gu, Tianrong Chen, Ying Shen

本文提出归一化轨迹模型（NTM），旨在解决扩散模型在极少步数（如4步）采样下因高斯去噪假设失效而导致的性能下降问题。NTM将每一步逆向生成建模为可精确计算似然的条件归一化流，通过浅层可逆模块与跨轨迹的深层并行预测器协同架构，实现端到端训练或从预训练流匹配模型初始化。其精确的轨迹似然还支持自蒸馏：仅用四步即可训练出轻量级去噪器，生成高质量图像。实验表明，NTM在文本到图像生成任务中，以仅4步采样即达到或超越主流生成模型性能，同时唯一保留了对完整生成轨迹的精确似然估计。

4. SCOPE: Structured Decomposition and Conditional Skill Orchestration for Complex Image Generation#

Tianfei Ren, Zhipeng Yan, Yiming Zhao

本文针对文本到图像生成中复杂视觉意图难以忠实实现的问题，提出SCOPE框架，旨在解决语义承诺（semantic commitments）在生成全生命周期中因概念断裂（Conceptual Rift）导致的跟踪失效问题。SCOPE通过结构化规格演进显式建模语义承诺，并基于其状态条件调用检索、推理与修复技能。为评估承诺级意图实现能力，作者构建了人工标注基准Gen-Arena及严格指标EGIP。实验表明，SCOPE在Gen-Arena上EGIP达0.60，显著优于基线，并在WISE-V（0.907）和MindBench（0.61）上表现优异，验证了持久化承诺追踪对复杂图像生成的有效性。

5. Fast Byte Latent Transformer#

Julie Kallini, Artidoro Pagnoni, Tomasz Limisiewicz

本文提出Fast Byte Latent Transformer（FBLT），旨在解决字节级语言模型（LMs）因逐字节自回归生成导致的推理速度瓶颈问题。作者设计了三种高效生成方法：（1）BLT-Diffusion（BLT-D），引入块级扩散辅助目标，支持每步并行生成多个字节；（2）BLT Self-speculation（BLT-S），利用局部解码器越界“起草”字节并由全模型单次验证；（3）BLT-Diffusion+Verification（BLT-DV），在扩散生成后增加自回归校验步骤。实验表明，三者均可将内存带宽开销降低50%以上，在保持字节级建模优势的同时显著提升推理效率与生成质量，推动字节级LM走向实用化。

6. Conformal Path Reasoning: Trustworthy Knowledge Graph Question Answering via Path-Level Calibration#

Shuhang Lin, Chuhao Zhou, Xiao Lin

知识图谱问答（KGQA）虽具可解释性优势，但现有方法难以提供可靠的答案覆盖保证。针对传统共形预测（CP）在KGQA中校准有效性与分数区分度不足的问题，本文提出共形路径推理（CPR）框架：其一，在查询级别对路径级得分实施共形校准，保障交换性并生成路径预测集；其二，设计轻量级残差共形值网络（RCVNet），通过PUCT引导的探索训练，学习高区分度的路径非共形性得分。在多个基准上的实验表明，CPR相较共形基线将实证覆盖率提升34%，同时平均预测集规模缩减40%，显著兼顾统计可靠性与答案紧凑性。

7. Zero-Shot Imagined Speech Decoding via Imagined-to-Listened MEG Mapping#

Maryam Maghsoudi, Shihab Shamma

本文提出一种零样本想象语音解码新方法，旨在克服非侵入式脑信号（如MEG）中想象语音数据稀缺、跨被试/会话时间对齐困难等挑战。研究采集了受过训练的音乐家在聆听与想象节奏性旋律及语音刺激时的配对MEG数据，利用其优异的时间一致性提升建模可靠性。方法采用三阶段解码流程：首先构建想象到聆听的MEG响应映射模型；其次仅基于聆听数据训练对比式词解码器，并评估多种语义、声学与音素嵌入策略；最后将新被试的想象MEG经映射后输入聆听解码器。秩分析表明，该方法可显著高于随机水平解码想象词汇。

8. EmambaIR: Efficient Visual State Space Model for Event-guided Image Reconstruction#

Wei Yu, Yunhang Qian

本文针对基于事件的图像重建任务中CNN难以建模全局依赖、ViT计算复杂度高（O(n²)）等问题，提出高效视觉状态空间模型EmambaIR。该模型融合事件流的时空稀疏性，创新性地设计了跨模态Top-k稀疏注意力模块（TSAM）与门控状态空间模块（GSSM）：TSAM实现像素级稀疏交互以生成紧凑融合特征，GSSM在保持线性复杂度（O(n)）的同时引入非线性门控机制增强时序建模能力。在运动去模糊、去雨和HDR增强三大任务共六个数据集上的实验表明，EmambaIR在重建质量上显著优于现有方法，同时大幅降低内存占用与计算开销。

1. GraphDC: A Divide-and-Conquer Multi-Agent System for Scalable Graph Algorithm Reasoning#

Wenjin Li, Jiaming Cui

本文针对大语言模型（LLMs）在图算法推理任务中性能不足的问题，提出GraphDC——一种基于分治策略的多智能体系统。该框架将输入图递归划分为子图，由专用代理并行执行局部推理，并通过主代理融合子图结果及跨子图信息，实现全局解的协同生成。其分层架构显著降低了单个代理的推理复杂度，缓解了计算瓶颈，提升了大规模图上的鲁棒性与可扩展性。实验表明，GraphDC在多种图算法任务上持续优于现有方法，尤其在大规模图实例上优势显著。

2. More Thinking, More Bias: Length-Driven Position Bias in Reasoning Models#

Xiao Wang

本文揭示了推理模型中一种被忽视的“长度驱动位置偏差”（Length-Driven Position Bias）：在多选题问答中，链式思维（CoT）推理轨迹越长，模型对选项位置（如A/B/C/D）的偏好越强，而非仅依赖内容判断。作者在MMLU、ARC-Challenge和GPQA上系统评估13种推理配置（涵盖7B至671B规模模型），发现12种存在显著正向偏相关（r=0.11–0.41, p<0.05）；截断实验证实该偏差具因果性——从推理中后段续写时，模型转向位置偏好答案的概率提升16%–32%。即使在671B模型中整体偏差微弱（PBS=0.019），最长推理 quartile 仍达0.071，表明准确率抑制而非消除该机制。研究指出CoT并非消除位置偏差，而是将其转化为与推理长度耦合的新形式，并提出PBS等可解释性工具以支持MCQ评估中的偏差审计。

3. Fast and Effective Redistricting Optimization via Composite-Move Tabu Search#

Hai Jin, Diansheng Guo

本文针对空间选区划分这一具有强连续性约束的组合优化问题，提出一种复合移动禁忌搜索算法（CM-Tabu）。该方法通过识别最小连通保持单元集或可交换单元对，构建满足连续性约束的复合移动操作，在不破坏解可行性前提下显著扩展邻域搜索空间。基于割点与双连通分量的线性时间图分析技术，高效生成单单元与复合候选移动。实验表明，CM-Tabu在解质量、运行稳定性与计算效率上均显著优于传统禁忌搜索及其他基线方法；在费城案例中可稳定达到人口均衡理论最优解，并支持多目标权衡，具备实际决策支持能力。

1. Domain-level metacognitive monitoring in frontier LLMs: A 33-model atlas#

Jon-Paul Cacioli

本文系统评估了33个前沿大语言模型在MMLU六大领域（应用/专业知识、形式推理、自然科学等）的领域级元认知监控能力，基于1500道题目（每域250题）与口头化置信度评分（0–100）计算Type-2 AUROC。结果表明：所有具备高于随机水平整体监控能力的模型均存在显著领域差异；应用/专业知识最易监控（平均AUROC=0.742），形式推理与自然科学最难（共占33模型中27个的末两位）；六领域划分具有实践合理性但非潜在结构验证；模型家族内监控剖面呈显著聚类（Anthropic、Gemini、Qwen），而DeepSeek、Gemma、OpenAI则不显著；Gemma 4较Gemma 3提升0.202 AUROC；二元KEEP/WITHDRAW探针失效模型在口头置信下仍表现正常，凸显探针格式特异性。研究揭示聚合指标掩盖的重要领域异质性，主张在部署前开展基准阶段的领域筛选。

2. VITA-QinYu: Expressive Spoken Language Model for Role-Playing and Singing#

Jiacheng Xu, Heting Gao, Liufei Xie, Zhenchuan Yang, Lijiang Li, Yiting Chen, Bin Zhang, Meng Chen, Chaoyu Fu, Weifeng Zhao, Wenjiang Zhou

本文提出VITA-QinYu，首个支持角色扮演与歌唱生成的端到端表达式语音语言模型（SLM）。针对语音中超越文本内容的个性、情绪及表演性表达建模难题，该模型采用融合文本与多码本音频token的混合语音-文本范式，在增强副语言信息表征能力的同时保障模态解耦。研究构建了覆盖15.8K小时自然对话、角色扮演与歌唱的高质量合成数据集。实验表明，VITA-QinYu在角色扮演客观评测上领先同类SLM 7个百分点，在歌唱主观MOS评分（5分制）上提升0.13分，并在C3和URO对话基准上分别以+1.38%和+4.98%的准确率与流畅度刷新SOTA。代码、模型及支持流式与全双工交互的演示系统均已开源。

3. IntentGrasp: A Comprehensive Benchmark for Intent Understanding#

Yuwei Yin, Chuyuan Li, Giuseppe Carenini

本文提出IntentGrasp——首个面向意图理解能力的综合性基准，涵盖12个领域、49个高质量开源语料，构建了包含26.3万训练样本、1.29万全量测试样本（All Set）及470个高难度均衡测试样本（Gem Set）的数据集。在20个主流大语言模型（含GPT-5.4、Gemini-3.1-Pro等前沿模型）上的评测显示，模型在All Set和Gem Set上平均F1仅低于60%和25%，其中17个模型在Gem Set上甚至低于随机猜测基线（15.2%），而人类表现达81.1%。为此，作者提出意向性微调（IFT）方法，在IntentGrasp训练集上微调显著提升性能（All Set +30+ F1，Gem Set +20+ F1），且留一域实验验证其优异的跨域泛化能力，为构建更智能、可靠与安全的AI助手提供了关键路径与坚实基础。

📄 arXiv: cs.LG#

1. RateQuant: Optimal Mixed-Precision KV Cache Quantization via Rate-Distortion Theory#

Fei Zuo, Zikang Zhou, Hao Cong, Xiaoyan Xi, Ho Fai Leung

本文针对大语言模型推理中KV缓存内存开销过大的问题，提出基于率失真理论的混合精度量化方法RateQuant。现有量化方案采用统一比特宽，忽视注意力头间重要性差异；而简单按重要性分配比特会导致“失真模型错配”——不同量化器的失真衰减速率β差异显著（3.6–5.3），跨模型复用失真模型反而劣于均匀量化。RateQuant通过小规模校准集为每种量化器拟合专属失真模型，并利用逆水填算法解析求解最优比特分配。在Qwen3-8B上，2.5平均比特下，其将KIVI的困惑度从49.3降至14.9（下降70%），优于QuaRot达6.6 PPL，校准仅需1.6秒且推理零开销。

2. LKV: End-to-End Learning of Head-wise Budgets and Token Selection for LLM KV Cache Eviction#

Enshuai Zhou, Yifan Hao, Chao Wang, Rui Zhang, Di Huang, Jiaming Guo, Xing Hu, Zidong Du, Qi Guo, Yunji Chen

本文针对大语言模型（LLM）长上下文推理中KV缓存内存随长度线性增长的瓶颈问题，提出端到端可学习的KV缓存淘汰框架LKV。LKV将KV压缩建模为可微优化问题，包含两部分：LKV-H用于学习任务驱动的头级全局预算，LKV-T则在不显式构建注意力矩阵的前提下，基于查询-键内在关系推断KV重要性。该方法摒弃依赖统计先验或固定归纳偏置的传统启发式策略，实现压缩目标与下游任务的严格对齐。在LongBench与RULER基准上，LKV在高压缩率下达到SOTA性能；尤其在LongBench上，仅保留15% KV缓存即可实现近无损推理。消融分析进一步证实，数据驱动的预算学习是性能提升的主导因素。
