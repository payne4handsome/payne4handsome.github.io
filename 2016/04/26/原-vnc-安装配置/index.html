[原]vnc 安装配置

发表于 2016-04-26

 安装vncserver网上有很多教程，但都会碰到很多问题，比如用vnc client连接桌面的时候什么都没有，或者只有一个terminal，光装一个vnc我就花了几个小时，下面给出我的配置方法，下面是装完的截图

一、安装桌面（安装其他桌面请用另外的配置方法）

sudoapt-get install gnome-core

二、安装vnc server

Apt-get install vnc4server

三、运行vncserver命令，会提示输入密码（此时还没有创建.vnc这个文件夹）

四、修改配置文件

#vim/root/.vnc/xstartup

复制下面的内容(要一模一样)

#!/bin/sh

def

exportXKL_XMODMAP_DISABLE=1

unsetSESSION_MANAGER

unsetDBUS_SESSION_BUS_ADDRESS

 gnome-panel &

gnome-settings-daemon&

metacity &

nautilus &

gnome-terminal &

五、创建连接

#vncserver 

一般从1开始……,然后你用vnc client连接就可以了
